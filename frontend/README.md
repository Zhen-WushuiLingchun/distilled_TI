# Distilled TI Frontend

当前前端提供三段体验：

- 首页：项目说明 + 可选 AI 配置
- 会话页：连续答题，20 题后可随时拿报告，也可继续作答
- 报告页：AI 摘要、雷达图、细节百分比条、模块投影
- 历史页：查看 TTL 生命周期内的本地会话
- 管理页：查看模板库、实例题、改写预览，并新建模板

## 本地运行

```bash
npm install
npm run dev
```

默认访问：

- `http://127.0.0.1:3000`

## 环境变量

可选设置：

```bash
NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8000/api
```

如果不设置，默认也会请求上面的本地后端地址。

## 说明

- 报告页依赖后端 `20` 题门槛。
- 若首页没有配置 AI，报告文字会使用后端降级总结。
- 若要联调，请先启动 backend，再启动 frontend。
