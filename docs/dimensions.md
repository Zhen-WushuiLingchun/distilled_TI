# Distilled TI Dimensions

## Core Dimensions

1. `social_initiative`
   - 定义：主动开启互动、发起协作、率先表达的倾向。
   - 高分表现：更常主动联络、组织、开场。
   - 低分表现：更常等待情境成熟后再进入。

2. `social_stimulation_tolerance`
   - 定义：在高密度互动、噪声和刺激下维持状态的能力。
   - 高分表现：多人环境中仍能稳定互动。
   - 低分表现：高刺激下更容易消耗或退出。

3. `autonomous_judgment`
   - 定义：偏独立判断还是偏依赖权威、流程、共识。
   - 高分表现：更愿意先形成自己的判断。
   - 低分表现：更依赖外部框架与共识。

4. `planning_preference`
   - 定义：对清单、结构化、预先规划的偏好。
   - 高分表现：更喜欢先排计划再推进。
   - 低分表现：更偏边做边调。

5. `risk_tolerance`
   - 定义：面对不确定性时承受波动与下注的倾向。
   - 高分表现：更愿意尝试高波动方案。
   - 低分表现：更偏稳健低风险路径。

6. `abstraction_tendency`
   - 定义：偏好概念、模型、理论还是实例、经验、操作。
   - 高分表现：先抓抽象结构与原则。
   - 低分表现：先看案例与可执行细节。

7. `novelty_seeking`
   - 定义：对新方法、新环境、新体验的追求程度。
   - 高分表现：主动寻找新鲜路径。
   - 低分表现：更偏熟悉和验证过的方法。

8. `competition_cooperation`
   - 定义：默认驱动更偏竞争领先还是合作共建。
   - 高分表现：更在意领先、差异化和胜出。
   - 低分表现：更在意协作、共识和整体完成。

9. `emotional_stability`
   - 定义：受反馈、冲突与扰动影响的强弱和恢复速度。
   - 高分表现：波动小、恢复快。
   - 低分表现：更容易被外界扰动带走状态。

10. `execution_drive`
    - 定义：把判断转成推进、落地和收尾的能力倾向。
    - 高分表现：更容易启动并持续推进到完成。
    - 低分表现：更容易停留在想法或准备阶段。
